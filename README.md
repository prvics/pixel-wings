# pixel wings
A fast-paced mobile shoot'em up by modern bullet hell games like Acecraft.

## Tech Stack
- Phaser
- Vite
- TypeScript

## Current Status
Early prototype:
- Player movement
- Shooting system
- Basic game loop

## Getting started
```bash
npm install 
npm run dev
```
Then open in your browser or mobile device:
```bash
http://localhost:5173
```
## Vision
pixel wings aims to became a polished mobile shooter with:
- Boss fights
- Upgrade system
- Roguelike elements
- Skill-based gameplay
